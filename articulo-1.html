Historia de CSS

El nacimiento del CSS (Cascading Style Sheets). CSS nació en 1996 como una propuesta de Håkon Wium Lie mientras trabajaba con Tim Berners-Lee, el creador de la web. Antes de CSS, los estilos como colores, tamaños y márgenes tenían que escribirse directamente en el código HTML, lo que hacía las páginas difíciles de mantener.










¿Qué hace CSS?

CSS permite separar el contenido (HTML) del diseño visual. Gracias a CSS, se puede cambiar la apariencia de múltiples páginas web desde un solo archivo, controlando fuentes, colores, alineaciones, márgenes, y más.

Evolución de CSS

  • CSS1 (1996): Introdujo estilos básicos como colores, márgenes y tipografías.
  • CSS2 (1998): Añadió control sobre layouts, posiciones y soporte para impresión.
  • CSS3 (2005+): Se dividió en módulos y permitió transiciones, animaciones y diseños responsivos.
